# Distributed-file-storage-system
Full-stack file storage system using Spring Boot and React with JWT authentication, file upload/download, and REST APIs (work in progress)

## Features
- JWT Authentication (in progress)
- File Upload, Listing, Download
- REST APIs
- User validation

## Tech Stack
- Java, Spring Boot
- React
- MySQL

Project is currently under development. Code will be updated shortly.
